# taxes-automation-kurs1
Целью проекта являлась разработка приложения (Windows Forms) «Автоматизация учета и контроля полноты и своевременности уплаты налогов, сборов и платежей» в среде MS Visual Studio 2022 на языке C#.

Объект исследования – процесс управления налоговыми обязательствами физических и юридических лиц.
Предмет исследования – объектно-ориентированные средства языка программирования C# для повышения эффективности учета и контроля налоговых платежей.

Разработанная программа позволяет рассчитать размер необходимых выплат, разделенных по категориям, а также предоставляет возможность перейти на сайт Федеральной налоговой службы для оплаты. Предварительно пользователю дана возможность выбрать аккаунт, а далее – просмотреть данные, связанные с выбранным аккаунтом.

## Что реализовано:
1. Аутентификация и авторизация на простейшем уровне - я предусмотрела три тестовых аккаунта, каждому из которых присвоена уникальная информация о пользователе, пароль и личные данные.
2. База данных реализована в качестве глобального статического класса. Можно считать это заглушкой для реальной базы данных.
3. Возможность просмотра и рассчета ряда налогов, разделенных на две категории: для физических и для юридических лиц.
4. User-friendly оформление - понятная навигация и приятная глазу цветовая гамма, применение различных элементов (заголовки, обычный текст, списки, выпадающие списки, кнопки, ссылки, меню, текстовые поля). Эта часть реализована настолько, насколько возможно, средствами C# и .NET. К сожалению, в данном случае возможности Windows-форм ограничены в плане визуализаций.
5. Оплата предусмотрена на официальном сайте ФНС, поэтому ссылка перенаправляет на сайт.
6. Просмотр информации об аккаунте в отдельном окне
7. Для удобства дополнительно интерактивными элементами реализованы функции выхода из аккаунта и завершения работы программы.
### Как протестировать:
Скачать папку release, запустить exe-файл. Exe-файл можно найти по следующему пути: kursovaya\kursovaya\bin\Release\kursovaya.exe

Пароли для входа в аккаунты:
```
Account1.Password = "123";
Account2.Password = "234";
Account3.Password = "345";
```
